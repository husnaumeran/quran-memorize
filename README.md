# Quran Memorize

A web app for Quran memorization using a progressive repetition pattern.

## Memorization Pattern

- Verse 1 × N times
- Verse 2 × N times, then verses 1-2 together × N times
- Verse 3 × N times, then verses 1-2-3 together × N times
- And so on...

## Setup

```bash
pip install -r requirements.txt
python main.py
```

Then open http://localhost:8000

## Features

- Arabic text with proper RTL display
- Audio playback for each verse
- Configurable chapter, verse range, and repetitions
- Progressive memorization pattern
